• The basics of SQL

    •  SELECT x, FROM table, WHERE condition


• There are also WILDCATS

    •  %:

       •  Example: WHERE name like "%a",

       •  This means there are any amount, of any letter before a,

       •  So the word (name in this case) finishes in an "a".

    •  _ :

       •  Example: WHERE name like "%a_"

       •  The underscore means that there is one letter after the a,

       •  The percentage means the same as before.

       •  a has to be the second to last character.
